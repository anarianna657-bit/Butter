Butter is a mobile game where you can make butter from scratch in your own personal bakery and sell it to customers for in game currency in which you can use to buy in game items.

---

## Tech Stack

- **Framework:** React Native (Expo)
- **Language:** TypeScript
- **Navigation:** React Navigation
- **State Management:** React Context (initial) / Zustand (later)
- **Storage:** AsyncStorage (initial) / SQLite (later if needed)
- **Testing (later):** Jest + React Native Testing Library

---

## Project Goals

### MVP (Must do's)
- [ ] Core feature #1: Make butter
- [ ] Core feature #2: Sell buttered bread to customers
- [ ] Core feature #3: Get coins
